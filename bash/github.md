# Creating, editing, and merging in branches

- `git pull https://github.com/pointOfive/Examples.git`
- `git checkout -b clone_to_edit`
- `git status`
- `git add github.md`
- `git commit -m 'updating a file'`
- `git branch`
- `git checkout master`
- `git merge clone_to_edit`
- `git branch -d clone_to_edit`
- `git push`
